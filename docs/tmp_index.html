About

RDSim is a Robo Delivery Simulator developed for autonomous delivery systems. It integrates state-of-the-art SLAM, localization, planning, and control technologies within the Gazebo simulation environment. Designed as a comprehensive solution, RDSim supports robot control, environment simulation, and robust navigation capabilities.

    • ROS2 control

  •   (2024.10.) Advanced planning and control with MPPI controller tuning, topology map navigation, and GPS+IMU fusion.
    • MPPI controller parameter turning
    • Nav2 Topology map navigation

    • 3D LiDAR SLAM (GLIM)
    • Nav2 Package Configuation
    • GLIM factor graph convert to pcd file
    • Localization (HDL localization) & Integrate with Nav2
  •   (2024.04.) Initial development of RDSim with Gazebo simulator and Docker container support.
    • Initialize RDSim based on CitySim
    • Basic Robot control
    • Setting Docker container
    • Gazebo dynamic actor based on gazebo_sfm_plugin
